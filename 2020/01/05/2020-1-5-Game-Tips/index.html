Add Bots

Let’s talk about how to load the robot BOT:

After entering the game, select your faction. After you get the gun steady, open the console and enter:forceaddbots botsnumber camp

Soviet camp = 1
German camp = 2

for example:
The Soviets adds 2 soldiers, in console:forceaddbots 2 1

The German adds 3 soldiers, in console:forceaddbots 3 2

Rising Storm 2: Vietnam

To play single-player mode, after entering the game interface, press f3 to open the console.

Enter Open VNTE-CuChi?Minplayers=64?Maxplayers=64?Timelimit=1 200 to enter the map.

  • VNTE-mapname (no spaces when entering the map name)
  • minplayers = 64? maxplayers = 64 (minimum and maximum)
  • timelimit = 1 200 (1 is the number of rounds, 200 is the time)

After entering the map, enter addbots 64 and join the bot

Counter-Strike: Global Offensive

Press key ~ to open command first.

Cheat Mode

sv_cheats 0/1                -- disable/enable cheat(needed by folowing part)
god                         -- god mode
sv_infinite_ammo 0/1/2      -- limited/unlimited bullet/unlimited magzines
sv_grenade_trajectory 0/1   -- show trace of the projectile
sv_grenade_trajectory_time 10  -- show 10s
sv_showimpacts 0/1          off/show impact point after shooting
sv_showimpacts_time 1       -- show 1s

Display Setting

net_graph 0/1               -- display fps/ping/loss/ticks

Or add boot option -console -width 800 -height 600 to set resolution.

Or delete steamapp->common->Counter-Strike Global Offensivecsgo->cfg->video.txt, and enter game to reset.

Boot Options:

  • add boot option-perfectworld -language schinese to enter perfect world server and display chinese.

  • add boot option-tickrate 128 to enter 128 tick server.

  • after that, enter
    • cl_cmdrate 128
    • cl_updaterate 128
    • rate 786432

Config file

Commands can be saved in cfg file

As for cfg file, it is in steamapp->common->Counter-Strike Global Offensivecsgo->cfg->xxx.cfg.
Enter exec xxx.cfg in command to get it effect.
You can also add -console +exec xxx to boot option for convenience.
