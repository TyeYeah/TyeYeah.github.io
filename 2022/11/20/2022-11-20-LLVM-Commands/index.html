Actually LLVM compilation process includes following file formats:

  1. xxx.c, Source code.
  2. xxx.bc, Assembly LLVM bitcode
  3. xxx.ll, Textual LLVM bitcode
  4. xxx.s, Assembly code
  5. xxx.out, Executable
